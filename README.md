# SafeHome — премиальный лендинг по безопасности загородных домов

Проект демонстрирует одностраничный сайт для собственников коттеджей и вилл, которые покупают не камеры, а безопасность семьи, контроль и спокойствие.

## Структура проекта

- `index.html` — основной HTML лендинг
- `styles.css` — стили в тёмной premium стилистике
- `script.js` — логика калькулятора и лид-формы

## Как посмотреть

1. Откройте `videonablyudenie/index.html` в браузере.
2. Перейдите по ссылкам навигации для проверки разделов.
3. Заполните форму и рассчитайте стоимость проекта.

## Что изменено

- Эмоциональный первый экран с оффером «безопасность семьи».
- Блок выгод: бесплатный выезд, схема, смета, гарантия.
- Секция услуг с акцентом на результат, а не на оборудование.
- Кейсы с реальными цифрами: площадь, камеры, сроки, стоимость.
- FAQ с ключевыми возражениями собственников.
- Тёмная glassmorphism-стилистика, современные карточки и анимации.
- Адаптивная верстка для всех экранов.
